What Are Groupings?

A grouping collapses a table so there is one row per unique value in a column. Instead of seeing every individual request, you see a summary — one row per company, per month, per employee.

Common Use Cases

  • Revenue per quarter
  • Requests by company
  • Tasks by employee
  • Sales by region
Before (ungrouped)
Request 1Acme Ltd
Request 2Acme Ltd
Request 3Acme Ltd
Request 4Beta Co
Request 5Beta Co
After (grouped by company)
Acme Ltd3 requests
Beta Co2 requests

Tutorial 1: How to Group a Table

1
Click on the column you want to group by (e.g. Company Name)
2
Select Group Column from the dropdown menu
3
The table collapses to one row per unique value
4
Click + > Add Column to add a calculation
5
Type Sum([Revenue]) and press Enter
6
Rename the column "Total Revenue" by double-clicking the header

Key Function: Contains

Syntax

Contains(string, substring)
  • Returns True or False
  • Case-sensitive — wrap in Lower() to ignore case
  • Multiple substrings: returns True if any match
Flag Ltd companies
Contains([Company Name], "Ltd")
Case-insensitive match
Contains(Lower([Product]), "camera")
Multiple terms
Contains([Description], "urgent", "ASAP")

Text & Type Functions

Text Functions

  • Concat([A], " ", [B]) — join strings
  • Lower() / Upper() — change case
  • Trim() — remove leading/trailing spaces
  • Left([Col], 3) / Right([Col], 3) — extract chars
  • Len([Col]) — character count
  • Replace([Col], "old", "new") — substitute

Type Functions

  • Text([Col]) — convert to text
  • Number([Col]) — convert to number
  • Date([Col]) — convert to date
  • Logical([Col]) — convert to True / False

Aggregate Functions

Functions that summarise multiple rows into a single value — always used inside a grouped table.

Calculation

  • Sum([Col])
  • Avg([Col])
  • Count([Col])
  • CountDistinct([Col])
  • Min([Col])
  • Max([Col])
  • Median([Col])

Conditional

  • SumIf([Amount], [Status]="Paid")
  • CountIf([Col], condition)
  • AvgIf([Col], condition)
  • MaxIf([Col], condition)
  • MinIf([Col], condition)

Totals

  • PercentOfTotal(Sum([Revenue]))
  • GrandTotal(Sum([Revenue]))
  • Subtotal([Revenue])

Building Complex Formulas

1. Case-insensitive text check
Contains(Lower([Company Name]), "ltd")
2. Conditional sum on text
SumIf([Revenue], Contains(Lower([Category]), "software"))
3. % of total
Sum([Revenue]) / GrandTotal(Sum([Revenue]))
4. Period-over-period
Sum([Revenue]) - Lag(Sum([Revenue]), 1)
5. Tier label
If(Sum([Revenue]) > 10000, "High", If(Sum([Revenue]) > 5000, "Mid", "Low"))
